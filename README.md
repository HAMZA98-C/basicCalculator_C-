# 🧮 Basic Calculator in C++

This is a simple command-line calculator written in C++ that performs basic arithmetic operations: **Addition, Subtraction, Multiplication, and Division**.

## 📋 Features

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`) with division-by-zero check
- Clean and beginner-friendly code

## 🚀 Getting Started

### Prerequisites

To run this project, you need:

- A C++ compiler (e.g., `g++`, `clang++`, or any IDE like Code::Blocks, Dev C++, or Visual Studio)
